Connect Your Agent
Temporary shell access for AI agents. Create a session, run one command on the target machine, then paste the prompt into Claude Code, Codex, OpenClaw, or another AI agent.
Active sessions run in-memory only. Nothing is logged or stored.
Use Cases
  • Ask AI to check why your server is running out of disk space at 3am.
  • Ask AI to set up a new dev machine while you keep working.
  • Give AI temporary access to a throwaway machine, lab box, or recovery environment.
Supported: macOS Intel/Apple Silicon, Linux x64/arm64, Windows x64.
